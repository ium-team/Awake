# 로드맵

## Phase 0: 검증

- IOKit power assertion으로 일반 유휴 잠자기 방지 검증
- 디스플레이 꺼짐과 시스템 잠자기 차이 검증
- 덮개 닫힘 환경별 동작 기록
- 실행 중 앱/윈도우 목록 수집 방식 검증

## Phase 1: Core Product

- 메뉴바 앱 셸
- 대상 선택 화면
- 프로세스 단위 보호 세션
- 전원 assertion 생성/해제
- 대상 종료 감지와 자동 해제
- 권한/오류 안내

## Phase 2: 사용성 개선

- 전역 단축키
- 최근 선택 대상
- 즐겨찾기 앱
- 세션 남은 상태 표시
- 알림 개선
- 로그인 시 실행 옵션

## Phase 3: 정교한 감시

- 윈도우 단위 감시 개선
- 특정 프로세스 트리 감시
- 터미널 세션과 하위 프로세스 힌트
- 명령어 기반 세션 시작 옵션

## Phase 4: 고급 기능

- Shortcuts 연동
- CLI companion
- 앱별 프리셋
- 자동 시작 규칙
- 세션 히스토리와 에너지 사용 리포트
