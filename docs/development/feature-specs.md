# 기능별 개발 명세

## 메뉴바

- idle 상태: 기본 아이콘 표시
- active 상태: 강조 아이콘 또는 대상 수 표시
- 메뉴 항목:
  - `Keep Apps Awake...`
  - `Stop Keeping Awake`
  - 현재 세션 대상 요약
  - `Settings`
  - `Quit Awake`

## 대상 선택 화면

- 실행 중 앱 목록을 표시한다.
- 앱 이름, 아이콘, PID, 감지된 윈도우 수를 보여준다.
- 앱 행을 선택하면 해당 앱 전체를 보호 대상으로 둔다.
- 윈도우 목록은 가능한 경우 보조 정보로 보여준다.
- 여러 대상을 선택할 수 있다.
- `Start` 버튼은 하나 이상 선택해야 활성화된다.

## 보호 세션

- 세션 시작 시 assertion을 생성한다.
- 선택 대상 snapshot을 세션에 저장한다.
- 대상 생존 여부를 주기적으로 확인한다.
- 모든 대상이 종료되면 자동 종료한다.
- 사용자가 수동 종료하면 즉시 assertion을 해제한다.

## 알림

- 세션 시작 알림은 기본적으로 생략한다.
- 자동 종료 시 간단한 알림을 보낼 수 있다.
- assertion 생성 실패, 권한 부족, 감시 불가 상태는 UI에서 즉시 표시한다.

## 설정

- 체크 간격
- 디스플레이 잠자기까지 방지할지 여부
- closed-display best-effort assertion을 사용할지 여부
- 관리자 권한으로 lid-closed sleep disabled 모드를 사용할지 여부
- lid-closed 세션 중 실제 덮개 닫힘 이벤트에서 화면보호기를 실행할지 여부
- active 세션 중 전원 관련 설정 변경 시 즉시 전원 assertion에 반영
- 자동 종료 알림
- 메뉴바 active 표시
- 최근 대상 저장 여부
